## ✨ Features

| Feature | Description |
|---------|-------------|
| 👥 **Create Groups** | Add friends, roommates, or teammates to a group in seconds |
| 💸 **Split Expenses** | Split equally or set custom amounts — no math needed |
| 🤖 **AI Categories** | Claude AI auto-detects food, transport, shopping & more |
| 📊 **Clear Balances** | See exactly who owes what with minimum payments needed |
| 📈 **Analytics Dashboard** | Donut charts, spending breakdown, and member insights |
| 🎯 **Settlement Planner** | Optimized payment suggestions to settle up fast |
| 🔍 **Search & Filter** | Quickly find groups and expenses |
| 📱 **Responsive Design** | Works beautifully on mobile and desktop |

## ⚙️ Run Locally

```bash
# Clone the repository
git clone https://github.com/niraunjana/SplitEase.git

# Navigate to project folder
cd SplitEase/expense-splitter

# Install dependencies
npm install

# Start development server
npm run dev
```

Open **http://localhost:5173** in your browser.

## 🧠 How It Works

1. **Create a Group** — Name your group (e.g., "Goa Trip") and add members
2. **Add Expenses** — Enter description, amount, and who paid
3. **AI Categorizes** — Claude AI automatically tags each expense
4. **View Balances** — See a clear breakdown of who owes whom
5. **Settle Up** — Follow the optimized payment plan to settle with minimum transactions
